USAGE.PY v0.2.0 
import numpy as np
import tenso

# 1. Create a massive tensor
data = np.random.rand(1024, 1024).astype(np.float32)

# 2. Serialize to bytes
packet = tenso.dumps(data)

# 3. Deserialize (Instant)
restored = tenso.loads(packet)

print(f"Loaded: {restored.shape}")
# Output: Loaded: (1024, 1024)

Architecture

Tenso uses a minimalist, 4-part structure designed to be read directly from memory.

HEADER
8 Bytes
SHAPE
Variable
PADDING
0-63 Bytes
RAW BODY (C-Contiguous)
Payload

Zero-Copy

Maps memory directly from the buffer to NumPy. No CPU cycles wasted on parsing or copying data.

Secure

Strictly handles data structure. Unlike Pickle, Tenso cannot execute arbitrary code. Safe for public APIs.

Aligned

Protocol v2 enforces 64-byte memory alignment, unlocking AVX-512 vectorization optimizations.

Efficient

Deserialization burns 0% CPU. Free up your inference servers to run models, not parse JSON.
